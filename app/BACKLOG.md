## 🐛 バグ・既知の問題

| 優先度 | 内容 | 備考 |
|--------|------|------|
| 🟡 | PWA の Service Worker が古いキャッシュを返すことがある | ビルド後にリロードしても古いUIが残るケース |
| 🟡 | dev server がワークツリー側で起動している場合、メイン側の編集が自動反映されない | `cp` で手動同期が必要 |
| ⚪ | Web Speech API の音質・速度がブラウザ・OSによってばらつく | Chrome 推奨だが Safari でも動作はする |
| ⚪ | `InputMode` の採点が完全一致のみ（ウムラウト入力ミスも不正解扱い） | `ü → ue` などの代替表記を許容するか検討 |
