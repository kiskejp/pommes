Pommes — プライバシーポリシー

収集するデータ

本アプリはいかなる個人情報も収集しません。学習の進捗・苦手問題リストはお使いの端末の localStorage にのみ保存され、外部サーバーには送信されません。

使用する API・外部サービス

  • Web Speech API — 日本語・ドイツ語の音声読み上げに使用します。音声合成はお使いのブラウザまたは OS が処理します。音声データは外部に送信されません。
  • Google Fonts — フォントの読み込みに使用します。PWA インストール後はキャッシュから読み込まれます。

第三者への提供

利用者の情報を第三者に提供することは一切ありません。

アクセス解析

本アプリはアクセス解析ツールを使用していません。
