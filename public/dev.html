Completion画面の再現について
セッション状態（ok / ng / ngIds）は React のメモリ内 state（useSession）で管理されており、 localStorage / sessionStorage 経由では注入できません。

再現方法：アプリで実際に数問解いて完了画面に進むか、
useSession.js の初期値を一時的に書き換えてデバッグしてください。
オフライン状態の模擬
Chrome DevTools → Network タブ → 「No throttling」 を Offline に切り替えてください。
Service Worker がキャッシュから応答し、アプリ上部にオフラインバナーが表示されます。
